Transfering a file from MacOS to Android or Android to MacOS is Hell. BreadChan is web application enabling file transfert between 2 🌐browsers windows. Working on 💻Desktop and 🤖Android. Source available here: https://github.com/AlexSalvatorio/breadchan.

The Breadchan's server doesn't host kind of file. Files are hosted on the user desktop and transfered using the webRTC canal. Also we use public torrent trackers to connect users, so we do not recommand sharing any kind of sensitive data.
